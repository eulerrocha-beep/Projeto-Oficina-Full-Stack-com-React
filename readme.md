# 🚗 Sistema de Gestão para Oficina Mecânica

Esse projeto foi criado com o intuito de praticar React, Node.js, Express e diversos conceitos do desenvolvimento Front-end e Back-end.

A ideia foi desenvolver um sistema para oficina mecânica onde é possível cadastrar clientes, veículos e ordens de serviço, além de ter um dashboard com informações e métricas do sistema.

---

# 🚀 Funcionalidades

✅ Cadastro de clientes  
✅ Cadastro de veículos  
✅ Cadastro de ordens de serviço  
✅ Atualização de status da OS  
✅ Exclusão de ordens  
✅ Dashboard com métricas  
✅ Gráfico de faturamento  
✅ Indicadores de crescimento mensal 📈📉  
✅ Sistema de login/autenticação  

---
